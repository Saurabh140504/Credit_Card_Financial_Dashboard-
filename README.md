# 📊 Credit_Card_Financial_Dashboard

👤 Author
Saurabh Gopal Chaudhari

📌 Project Overview

This project focuses on building an interactive Credit Card Financial Dashboard using Power BI to analyze customer behavior and transaction patterns.
The dashboard provides clear insights into revenue trends, customer segmentation, and spending habits to support better business decisions.

🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query (Data Cleaning)
* DAX (Data Analysis Expressions)
* Excel / CSV Dataset

🎯 Project Objective

* The goal of this project is to:
* Track revenue and transaction performance
* Understand customer demographics and behavior
* Analyze spending patterns across categories
* Help stakeholders make data-driven decisions

📂 Dataset Information

🔹 Credit Card
* Age Group
* Gender
* Job Type
* Education Level
* Income Group

🔹 Customer 
* Transaction Amount
* Transaction Count
* Revenue & Interest Earned
* Expenditure Type (Bills, Fuel, Travel, etc.)
* Card Category (Silver, Gold, Platinum, Blue)

⚙️ Steps to Build the Dashboard

Data Collection
* Imported CSV files into Power BI
  
Data Cleaning (Power Query)
* Removed missing values
* Fixed data types
* Renamed columns
  
Data Modeling
* Created relationships between tables

DAX Measures Created
* Total Revenue
* Total Interest
* Transaction Count
* Total Income
  
Dashboard Development
* KPI Cards for key metrics
* Line chart for revenue trend
* Bar charts for segmentation
* Slicers (Quarter, Gender, Card Type)

📊 Dashboard Insights

💳 Credit Card Transaction Report
 * Key Findings
 * Total Revenue: 55M
 * Interest Earned: 7.8M
 * Total Transactions: 656K
 * Q3 shows the highest revenue (~14.2M)
 * Transaction volume increases along with revenue
 * Highest spending category: Bills (14M)
 * Other major categories: Entertainment, Fuel, Grocery
 * Businessman segment contributes the highest revenue
 * Graduates generate the highest transaction value
 * Blue card category dominates revenue contribution
 * Customer acquisition cost is highest for Blue cards

👉 Insight Summary:
* Customers actively use credit cards for daily essential expenses, and business professionals drive the majority of revenue.
  
🧾 Credit Card Customer Report
 * Key Findings
 * Total revenue reached 55M, showing strong overall performance
 * Male customers contribute more revenue than female customers
 * Age group 40–50 is the highest revenue-generating segment
 * High-income customers generate the maximum revenue (22M)
 * Married customers spend more compared to single customers
 * Top-performing states: TX, NY, CA
 * Swipe transactions dominate over online usage
 * Businessman & White-collar jobs contribute the highest revenue
 * Graduates generate the most revenue among education groups

👉 Insight Summary:
* The business mainly depends on middle-aged, high-income, and professional customers, making them the most valuable target segment.
